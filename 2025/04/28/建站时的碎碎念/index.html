建站时的碎碎念

其实搭建博客的这个想法并不是最近才有的，早在 2023 年 10 月份我就开始尝试搭建博客，也同样是基于 Hexo 和 Github 的，那为什么现在才建成呢，原因是因为当时我完全对这个东西一窍不通，不可能总是问同学，而且搭建出来的博客也没有后台，再加上当时竞赛的压力也就把这个项目烂尾了。

前两天我想要重启这个项目，同时发现 LLM 很厉害，于是在 LLM 的帮助下成功搭建起了自己的博客，也在 LLM 的指导下学会了如何去管理这个网站，令人不禁感慨科技进步之快，现在我有任何想要升级这个网站的想法都可以直接咨询 LLM。

现在我还没有想清楚要往这个网站上投递一些什么内容，学术内容肯定是会有的，会发一些我感兴趣的算法领域的小研究。也有把我的回忆录迁移到这个博客上的想法，可能还会发一些日常见闻、评论、照片和文化课内容，反正就是我想发什么就可以发什么，算是有了一个自己思想上的小窝吧。

就这样，我要去上课了，再写多了就赶不上英语课了，希望这个博客的内容能一天天充盈起来吧！
